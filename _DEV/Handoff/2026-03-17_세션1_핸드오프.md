### 참가신청 폼 (`/apply`)
- 3단계 멀티스텝 폼 (StepIndicator 진행바)
  - Step 1: 신청 안내 확인 동의
  - Step 2: 프로그램 선택 → 동적 필드 (팀/개인, 보호자, 지도교사)
  - Step 3: 대회 운영 규정 동의 + 개인정보 수집 동의 → 제출
- Supabase `applications` 테이블 INSERT 연동
- programs slug → UUID 조회 후 저장
- 공통 UI 컴포넌트: Input, Select, Checkbox

### 어드민 (`/admin`)
- 1280px 와이드 레이아웃 (240px 사이드바 + 콘텐츠)
- 유저앱 레이아웃과 완전 분리 (#root 제약 없음)
- `/admin` — 대시보드 (통계 카드 4개)
- `/admin/applications` — 참가신청 관리 (상태 필터, 승인/반려)
- `/admin/programs` — 프로그램 관리 (뼈대)
- `/admin/notices` — 공지사항 관리 (뼈대)

### 테마/스타일 시스템
- CSS Modules (인라인 스타일 금지)
- 토큰 기반 (`src/styles/theme/tokens.css`)
- font-size/spacing: px 직접값 + clamp() (320px~600px 스케일링)
- font-weight: 숫자값 (400~700)
- color: hex 직접값
- 폰트: Pretendard Variable CDN 단일 사용
- 아이콘: @heroicons/react 단일 사용
