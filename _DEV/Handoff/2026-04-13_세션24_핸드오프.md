## 1. 세션 24 에서 끝난 것

### A. 부스 카드 시간 버튼 → footer 오버레이 방식
- 기존: footer 2줄 구조 (1줄 금액+버튼, 2줄 시간 그리드) → 버튼 클릭 시 2줄째 사라지며 레이아웃 점프 발생
- 변경: footer 1줄 구조 복원. waiting 상태에서 시간 써클 버튼 5개(5/10/15/20/30) + 거절 버튼이 footer 영역에 반투명 오버레이(`rgba(255,255,255,0.7)`)로 표시
- 오버레이는 우측 정렬, 클릭 시 오버레이만 사라지고 카드 높이 변화 없음
- 주문 내역(메뉴, 금액)이 오버레이 뒤로 비쳐서 시간 판단 가능

### B. 거절 버튼 양쪽 단계 모두 노출
- waiting 오버레이: 거절 + 시간 버튼 5개
- inProgress: 거절 + 조리완료 (기존처럼)
- 어느 단계에서든 거절 가능

### C. 거절 버튼 스타일 변경
- 기존: 흰 배경 + 빨간 테두리/텍스트
- 변경: 빨간 배경(`#DC2626`) + 흰 텍스트 — 직관적

### D. 완료 카드 정보 추가
