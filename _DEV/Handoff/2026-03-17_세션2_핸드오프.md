### 참가신청 폼 리팩터링 (`/apply`, `/apply/:slug`)
- **탭 네비게이션** — 4개 프로그램 탭 (백일장, 사생대회, 댄스, 합창)
- **프로그램별 독립 폼** — 각 프로그램 고유 필드 + 스텝 수 다름
  - 백일장 (2스텝): 참가정보 → 부문/작품유형 + 동의 + 제출
  - 사생대회 (2스텝): 참가정보 → 부문 + 동의 + 제출
  - 댄스 (3스텝): 기존 범용 폼 재활용 (programId 프리셋)
  - 합창 (4스텝): 안내확인 → 합창단정보 → 참가곡2곡 → 동의 + 제출
- **진입 경로 2개**: `/apply` 탭 선택 / `/program/:slug`에서 직접 이동
- 프로그램별 고유 데이터는 `meta` JSON 컬럼에 저장
- **RadioGroup** UI 컴포넌트 신규 생성 (성별, 부문, 구성 등 radio 선택)
