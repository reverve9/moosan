# Handoff — 2026-04-08 세션 9

이번 세션은 **매장(부스) 운영 앱 + 어드민 실시간 모니터 풀스택** 한 묶음 완성. 알람 묶음(Wake Lock/소리/진동/탭 타이틀)은 의도적으로 후순위로 미룸. 세션 8 까지 사용자 측 + Phase 1.9 + 토스 모바일 검증 다 끝난 위에 매장 측을 통째로 얹은 세션.

세션 8 결정사항이었던 "매장 dashboard (Phase 2B~2F) 보류 → 태블릿 별도 앱" 을 세션 9 시작 시점에 사용자가 뒤집음. 이유: **운영 당일 무조건 필요한 페어 시스템**이고 (운영요원 2명이 25개 부스 주문 확인을 대신 처리하는 건 비현실적), 컨텍스트 fresh 한 시점에 같이 끝내는 게 리스크 최저. 시나리오는 v1+v2+v4 합쳐 다 정해져 있었고, 알람만 빼면 디바이스 없이도 와이드 폼팩터 UI/기능 전부 구현 가능.
