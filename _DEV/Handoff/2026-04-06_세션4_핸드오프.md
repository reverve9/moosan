## 완료 사항

### YouthPage — 청소년문화축전 페이지 (`/program/youth`)

**좌측 포스터 + 우측 본문 about 섹션**
- `grid-template-columns: 2fr 3fr`, `align-items: center` (좌측 포스터를 우측 영역 기준 세로 센터)
- 본문: 드롭캡 박스(정사각형 2.8em × 2.8em) + 2단락 설명 + 정보 박스(행사기간/장소)
- 정보 박스: padding 12px 24px, row-gap 4px, 컴팩트
- 페이지 상단 padding `14px` (페이지 타이틀과의 간격 미세 조정)
- 드롭캡 박스 + 정보 박스 배경: `var(--festival-tint)` (페이지 스코프 변수)
- `text-align: justify` + `word-break: break-all` (한 글자 단위 줄바꿈으로 우측 라인 정렬)

**드롭캡 처리 — 핵심 포인트**
- `::first-letter` 의사요소는 **block container box에만 적용**되므로 `display: inline-flex` (float blockify되어도 flex container)에서는 무시됨
- 해결: 글자를 별도 자식 span(`.dropCapChar`)으로 감싸 직접 `font-size`를 적용
- 박스 사이즈 `2.8em`, 내부 글자 `2em` (마지막 확정 값)

**테마 컬러 시스템**
- `.page`에 페이지 스코프 CSS 변수 정의:
  - `--festival-tint: #FBF1CC` (옅은 옐로 — 박스 배경)
  - `--festival-accent: #E8B935` (진한 옐로 — 버튼 배경)
- 동적 페이지(DB fetch 시)는 `theme_color`로 `--festival-tint` 인라인 오버라이드, accent는 CSS fallback 유지

### ProgramAccordion 컴포넌트 (`src/components/program/`)
**닫힘 상태**: 썸네일(84×84) + 행사명 + 1-3줄 설명 + chevron
- 썸네일은 `<img object-fit:cover>`, `_DEV/design/`에서 4종 PNG 복사 → `public/images/thumb_*.png`
- 설명은 `-webkit-line-clamp` 제거 → 자연스러운 줄바꿈, 전체 표시

**열림 상태**: 통합 상세 박스 + 참가신청 버튼
- 통합 박스(`detailBox`) — 정보 박스 + 시상 박스를 하나로 병합 (구분선 없이 라벨-값 grid)
  - 6개 필드: `행사명 / 일 시 / 장 소 / 참가대상 / 시상내용 / 참가신청`
  - `참가신청` 행은 disc 불릿 리스트 (`접수기간` + `접수방법` 2개 sub-item)
  - 배경: `var(--color-bg-secondary)` (#F5F0E8 베이지 — 상단 옐로 박스와 차별)
  - `white-space: pre-line` 적용 → `\n` 기반 줄바꿈 디스플레이 (예: 합창대회 참가대상)

**참가신청 버튼**
- `Button` 컴포넌트 대신 `<Link>` 직접 사용 + 커스텀 클래스
- 배경 `var(--festival-accent)`, 글자 `var(--color-text)`, min-width 168px, padding 12px 30px
- 호버: opacity 0.88, 클릭: scale(0.98)
- 페이지 테마 컬러를 자동 상속 → 무산/음식 페이지 만들 때 변수만 다르게 설정하면 됨

**아코디언 동작**

- 같은 카드 헤더 다시 클릭 시 닫힘
- 펼침 애니메이션: `grid-template-rows: 0fr → 1fr` CSS-only 300ms ease

### 4개 프로그램 데이터 (현재 ProgramAccordion 하드코딩)
순서: **사생대회 → 합창대회 → 백일장 → 스트리트댄스**

| 표시명 (`name`) | 일정 | 장소 |
|---|---|---|
| 전국 어린이 사생대회 | 2025년 5월 24일(토) 오후 1시 ~ 3시 30분 | 청호호수공원 / 엑스포광장 |
| 전국 어린이 합창대회 | 2025년 5월 23일(금) 오후 5시 ~ 7시 30분 | 엑스포광장 특설무대 |
| 전국 청소년 백일장 | 2025년 5월 24일(토) 오후 2시 ~ 4시 30분 | 청호호수공원 / 엑스포광장 |
| 전국 청소년 스트리트댄스 페스티벌 | 2025년 5월 25일(일) 오후 5시 ~ 7시 | 엑스포광장 특설무대 |

- 각 프로그램은 `eventName`, `description`, `target`, `awardsText`, `registrationPeriod`, `applicationMethod` 필드 보유
- 사생대회는 첫 스크린샷 기준 정확한 데이터, 나머지 3개도 추가 스크린샷 기준으로 채움

- **`festivals` 테이블 신규** (3개 메인 페이지 메타) — slug, name, subtitle, description_lead/body, poster_url, schedule, venue, theme_color, sort_order
- **시드 3건**: musan(#E1F0DC) / food(#FBE3D8) / youth(#FBF1CC) — 각 페이지의 본문/스케줄/베뉴/테마 컬러 포함
- **`AdminFestivals.tsx`** (`/admin/festivals`) — 3개 카드:

  - 우측: 행사명 / 영문 부제 / 일정 + 장소 / 테마 컬러(color picker) / 본문 첫 단락(textarea) / 본문 두 번째 단락(textarea) / 저장 버튼
- **YouthPage 동적 로딩** — `useEffect`에서 `fetchFestivalBySlug('youth')`로 fetch, `theme_color` → 인라인 style로 `--festival-tint` 주입, `description_lead` 첫 글자 자동 추출 → 드롭캡

### Programs 테이블 확장 + AdminPrograms 페이지

  - 우측: 표시명 / 카드 설명(textarea) / 행사명 / 일시 + 장소(2열) / 참가대상(textarea) / 시상내용(textarea) / 접수기간 / 접수방법(textarea) / 저장 버튼
