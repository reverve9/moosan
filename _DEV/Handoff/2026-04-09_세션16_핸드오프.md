### D. 후원 텍스트 → 이미지 두 줄 (핵심)

**문제**: "주최·주관 / 후원" 영역을 푸터에 표시해야 하는데, 텍스트 리스트(문화체육관광부 / 강원특별자치도 / 속초시) 만 있던 상태였음. 사용자 요청은 **로고 이미지 사용** + **두 줄로 분리**.

**발견 과정** — `home_logo.png` 분석

`public/images/home_logo.png` 가 1200×1652 흰색-on-투명 PNG. 그냥 보면 빈 화면. PIL 로 검은 배경 합성 후 분석:

```
content bbox: (23, 1468, 1177, 1523)
clusters in row (x ranges):
  (23, 423)    ← 주최·주관 + 재단법인 설악·만해 사상실천선양회
  (526, 821)   ← 후원 + 문화체육관광부   ← 최대 갭(103px) 후 시작
  (871, 1034)  ← 강원특별자치도
  (1069, 1176) ← 속초 SOKCHO CITY
gap between cluster 1 ↔ 2: 103px ← split point
```

→ 원본은 1154×52 한 줄짜리 가로 스트립 (이미지 하단 y=1469~1521). 4개 클러스터가 가로로 나열돼 있고, 가장 큰 갭(`x=423~526`) 이 주최/주관 ↔ 후원 의 자연 split.

**스크립트 (PIL)** — `home_logo.png` 를 두 PNG 로 자름:

```python
img = Image.open('public/images/home_logo.png').convert('RGBA')
y0, y1 = 1463, 1527  # row + 6px pad
img.crop((13, y0, 433, y1)).save('public/images/footer_host.png')      # 420x64
img.crop((516, y0, 1186, y1)).save('public/images/footer_sponsors.png') # 670x64
```

**신규 에셋**
- `public/images/footer_host.png` (420×64) — 주최·주관 + 재단법인 설악만해사상실천선양회
- `public/images/footer_sponsors.png` (670×64) — 후원 + 문화체육관광부 + 강원특별자치도 + 속초

**렌더링 — 비율 보존이 핵심**

원본은 한 줄에 4개 클러스터가 같은 y-pixel-height(52px) 로 배치돼 있어서 logo 들의 시각적 크기가 일관됨. 두 PNG 로 잘라도 이 일관성을 유지하려면 **두 줄이 같은 스케일 팩터** 로 함께 줄어들/늘어나야 함.

```css
.sponsorRow {
  display: block;
  height: auto;
  filter: brightness(0);
  opacity: 0.5;
}
.sponsorRowHost     { width: 38%; }
.sponsorRowSponsors { width: 60.6%; }
```

- 380px 컨테이너 기준: host 144.4px, sponsors 230.3px → 둘 다 ~22px 톨
- 비율 검증: 38% / 60.6% = **0.627** = 원본 420 / 670 ✅
- `height: auto` 라 컨테이너가 커지면 (430px Pro Max) 둘 다 같은 비율로 fluid 하게 커짐
- 같은 height 고정값 (예: `height: 22px`) 으로도 비율은 보존되지만 fluid 스케일링이 안 됨 → % 너비가 정답

**.sponsorsBlock 위치 미세조정**
- `margin-left: -4px` — 푸터 좌패딩(`var(--space-5)` 20px) 안에서 4px 만 왼쪽으로 당김 (사용자가 시각적 정렬 요청)
- `margin-top: -10px` — 인스타(.contact 마지막) ↔ 후원 블록 사이 간격 16px → 6px 로 축소 (다른 섹션 간격은 그대로)
- `gap: 2px` — 두 줄 사이는 거의 붙임

### F. 카피라이트 2줄 + 컬러 톤다운

```tsx
<Text className={styles.copyrightText}>© Manhae Musan Foundation 2023 All Rights Reserved</Text>
<Text className={styles.copyrightText}>Produced by MGTNC 2026</Text>
```

- caption variant 가 기본 `<span>` 이라 인라인 흐름 → `.copyright { display: flex; flex-direction: column; gap: 1px }` 으로 컬럼 stacking
- `2023` 은 재단 설립 연도, `2026` 은 제작 연도 — 의도적으로 다름
- 컬러: `color="muted"` (`#999999`) 가 너무 흐려서 `.copyright .copyrightText { color: #7A7A7A }` 로 0,2,0 specificity 로 override
- 줄간격 1px (거의 붙임)
- 구분선 / 패딩은 원본 유지 (`margin-top: var(--space-2)`, `padding-top: var(--space-4)`, `border-top: 1px solid var(--color-border-light)`)

**다크 푸터 (#1C1815 웜 소프트 블랙)** — 사용자가 "한 번 해보자 안 이쁘면 원복" 으로 진행. 토큰 2개 (`--color-text-inverse-secondary`, `--color-text-inverse-muted`) 추가 + Text 컴포넌트 컬러 옵션 2개 추가 (`inverse-secondary`, `inverse-muted`) + 푸터 배경 / 디바이더 / 아이콘 컬러 인버스 적용까지 했다가 한번에 보고 원복 결정. 토큰/Text 변경분도 모두 원복함. **관련 변경 없음** (현재 main 상태 = 변경 전 상태).
