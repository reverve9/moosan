# 📋 PRD — 2026 설악무산문화축전 PWA

> 작성일: 2026-03-17
> 상태: 초안 (키비주얼 / 프로그램 세부정보 수령 후 보완 필요)
> 개발 도구: Claude Code

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 2026 설악무산문화축전 공식 앱 |
| 형태 | PWA (Progressive Web App) |
| 목적 | 축제 홍보 + 프로그램 참가신청 메인 플랫폼 |
| 행사일 | 2026년 5월 15일(금) ~ 17일(일) |
| 장소 | 강원도 속초시 청초호수공원 엑스포광장 |
| 주최/주관 | 설악만해사상실천선양회 |
| 후원 | 문화체육관광부, 강원특별자치도, 속초시 |

## 2. 타겟 유저

| 타겟 | 설명 |
|------|------|
| 1차 | 유·초등 학부모 (백일장, 사생대회 참가 희망) |
| 2차 | 초·중·고 학생 및 지도 교사 (댄스, 합창 참가 희망) |
| 3차 | 일반 축제 관람객 |

→ 모바일 퍼스트 필수. 학부모·학생이 스마트폰으로 접근하는 시나리오가 주.

## 3. 핵심 기능 요구사항

### 3-1. 페이지 구성 (싱글 페이지 앱, 섹션 스크롤 방식)

```
/ (메인)
├── #hero         메인 비주얼 + 행사 D-day 카운트다운
├── #about        축제 소개 (만해사상 / 행사 취지)
├── #programs     프로그램 목록 (카드형)
│   ├── /program/baekiljang     백일장 상세
│   ├── /program/saesaeng       사생대회 상세
│   ├── /program/dance          댄스경연 상세
│   └── /program/choir          합창대회 상세
├── #apply        참가신청 (프로그램 선택 → 폼)
├── #location     오시는 길 (카카오맵 or 네이버맵 삽입)
└── #notice       공지사항
```

### 3-2. 참가신청 폼 (앱 내 자체 구현)

**공통 필드:**
- 참가 프로그램 선택 (드롭다운)
- 참가자 이름
- 학교명 / 소속 기관
- 학년 / 나이
- 보호자 이름 (유·초 대상 프로그램)
- 연락처 (휴대폰)
- 이메일
- 개인정보 수집·이용 동의 (필수)

**제출 방식:**
- 백엔드 없이 운영 시 → Google Apps Script 웹훅으로 구글 시트에 저장
- 백엔드 있을 경우 → REST API POST 엔드포인트 연결
- ⚠️ 백엔드 여부 결정 후 확정 필요

### 3-3. PWA 요건
- `manifest.json` — 앱 이름, 아이콘, 테마 컬러
- `service-worker.js` — 오프라인 캐싱 (정적 자산)
- iOS Safari / Android Chrome 설치 지원
- OG 태그 — 카카오톡/인스타 공유 시 미리보기 이미지 표시

## 4. 디자인 방향

### 콘셉트
> "설악의 자연과 문화가 만나는 순간"
> 웅장한 설악산 + 청초호수의 자연미 + 예술·문화 축제의 생동감

### 컬러 팔레트 (키비주얼 확정 후 업데이트)
| 역할 | 색상 | 비고 |
|------|------|------|
| Primary | 딥 포레스트 그린 `#1E3A2F` 계열 | 설악산 |
| Accent | 골드/앰버 `#C9932A` 계열 | 문화/격조 |
| Background | 크림 화이트 or 다크 모드 | 키비주얼 보고 결정 |
| Text | 다크 차콜 | 가독성 |

> ⚠️ 키비주얼 수령 후 실제 컬러 추출하여 업데이트

### 폰트 (제안)
- 헤드라인: `Noto Serif KR` or `나눔명조` 계열 (한국 전통 문화 느낌)
- 본문: `Pretendard` (가독성)
- 영문/숫자: `Cormorant Garamond` or `Playfair Display` (격조)

### 무드
- 배경에 설악산/청초호수 사진 활용
- 섹션 전환 시 부드러운 페이드/슬라이드 애니메이션
- 후원기관 로고 푸터에 나열

## 5. 기술 스택 (제안)

| 영역 | 선택지 A (심플) | 선택지 B (풀스택) |
|------|----------------|-----------------|
| 프론트엔드 | Vanilla JS + HTML/CSS | React + Vite |
| 스타일 | CSS Variables + 순수 CSS | Tailwind CSS |
| 라우팅 | Hash 라우팅 | React Router |
| 폼 데이터 저장 | Google Apps Script 웹훅 | Node.js API + DB |
| 호스팅 | GitHub Pages / Netlify | Vercel / AWS |
| 지도 | 카카오맵 JS API | 동일 |

> 추천: **선택지 A** (기간 촉박, 유지보수 단순, PWA에 최적)
> 단, 참가신청 데이터 관리가 중요하다면 선택지 B 고려

---

## 6. 참가신청 데이터 흐름 (선택지 A 기준)

```
사용자 폼 제출
    ↓
fetch() → Google Apps Script URL (POST)
    ↓
구글 시트 자동 저장 (행 추가)
    ↓
담당자 이메일 알림 (선택)
```

→ 구글 시트로 모든 신청자 명단 실시간 확인 가능
→ 별도 서버 불필요

## 7. 페이지별 콘텐츠 명세

### #hero
- 배경: 키비주얼 이미지 (전면)
- 텍스트: `2026 설악무산문화축전`
- 날짜: `2026. 5. 15(금) — 17(일)`
- 장소: `강원도 속초 청초호수공원 엑스포광장`
- D-day 카운트다운 위젯
- CTA 버튼: "프로그램 보기" / "참가신청하기"
- 주최/후원 로고 바

### #about
- 축제 소개 텍스트 (만해 한용운 사상 기반 — 원고 추후 수령)
- 연혁 or 슬로건

### #programs (4개 카드)

| 프로그램 | 대상 | 세부정보 상태 |
|---------|------|-------------|
| 백일장 | 유치부 / 초등부 | ⏳ 이번주 중 수령 |
| 사생대회 | 유치부 / 초등부 | ⏳ 이번주 중 수령 |
| 댄스경연대회 | 초/중/고등부 | ⏳ 이번주 중 수령 |
| 합창대회 | 초/중/고등부 | ⏳ 이번주 중 수령 |

각 카드 클릭 시 → 상세 페이지 (참가자격, 접수기간, 시상내역, 주의사항)

### #apply
- 프로그램 탭 선택
- 자체 폼 (위 3-2 필드)
- 제출 완료 시 확인 메시지 + 이메일 안내

### #location
- 주소: 강원도 속초시 청초호수공원 엑스포광장
- 카카오맵 or 네이버맵 임베드
- 대중교통/주차 안내 텍스트 (추후 수령)

### #notice
- 공지 목록 (날짜 + 제목 + 내용)
- 초기 공지: 접수 시작, 주의사항 등
